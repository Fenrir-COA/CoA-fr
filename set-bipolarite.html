Set – Bipolarité

Bipolarité (Collier) (Epic II)

Attributs principaux

  • Neutralisation : +980
  • BATK : +13%
  • Dégâts de compétence : +13%
  • Tous attributs : +22
  • Attaque physique : +8%
  • Attaque Magique : +8%

Effets

[Mix]

Lorsque vous infligez des dégâts à un ennemi, déclenche [Cercle Equilibré] qui est un cercle de 8 mètres à votre position, durant 10 secondes (temps de recharge de 10 secondes, un seul cercle peut exister à la fois).

Tous les joueurs à l'intérieur du cercle gagnent :

  • Bonus de dégâts : 0.7% / seconde
  • Dégâts critiques : 1,8% / seconde
  • Dégâts aux ennemis sous debuffs : +12%

Le nombre maximum de stacks de base est 1.

Pour chaque équipement [Bipolarité] porté, le maximum de stacks est augmenté de 2.

Peut activer les sets , Divinité radieuse, Démoniaque et Brasier Solaire

Porter 3 sets différents ou plus annule tous les effets
