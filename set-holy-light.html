Set – Holy Light

Holy Light (Collier)

Attributs principaux

  • Neutralisation : +800
  • BATK : +5%
  • Dégâts de compétence : +5%
  • Tous attributs : +20
  • Taux de réduction de CD : +50
  • Charge de résonance : +6%

Effets

Hors résonance, toutes les 5 attaques critiques, gagne 1 cumul de [Explosion critique], cumulable jusqu’à 5.

En entrant en résonance, tous les cumuls sont convertis en [Explosion avancée].

Chaque cumul augmente de 8% les dégâts critiques de la prochaine compétence.

1 cumul est consommé par compétence utilisée.

Holy Light Set (Brassards)

Attributs principaux

  • Neutralisation : +800
  • Critique physique : +5%
  • Critique magique : +5%
  • Dégâts aux ennemis affaiblis : +6%
  • Recharge de résonance : +8%

Effets

Hors résonance, chaque compétence utilisée donne 1 cumul de [Compétence critique], jusqu’à 5.

En résonance, les cumuls deviennent [Compétence avancée].

Chaque cumul augmente les dégâts de compétence de la prochaine attaque de 5%.

1 cumul est consommé par compétence utilisée.

Holy Light Set (Anneau)

Attributs principaux

  • Neutralisation : +800
  • Bonus de dégâts : +8%
  • Dégâts critiques : +11%
  • Recharge de résonance : +6%
  • Vitesse d’attaque : +40

Effets

Hors résonance, toutes les 3 secondes, gagne 1 cumul de [Élément critique], jusqu’à 5.

En résonance, conversion en [Élément avancé].

Chaque cumul augmente l’amélioration de tous les attributs de +6 pour les prochaines compétences.

Les cumuls sont consommés lors de l’utilisation des compétences.

Holy Light Set (Talisman)

Attributs principaux

  • Neutralisation : +800
  • Amélioration de tous les attributs : +13
  • FOR : +9%
  • INT : +9%
  • Dégâts pendant la résonance : +6%

Effets

Chaque compétence utilisée donne 500 points d’énergie de résonance.

La valeur maximale est figée pendant la résonance.

  • Durée : 15 sec (consommé en 1 utilisation)
